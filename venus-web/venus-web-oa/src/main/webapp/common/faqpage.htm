如果希望首页是登录页面，请按如下步骤操作.
◎ 修改\WEB-INF\web.xml文件。 首先将默认欢迎页面配置为jsp/login/login.jsp，如下：
<!--The Welcome File List-->
<welcome-file-list>
<welcome-file>jsp/login/login.jsp</welcome-file>
</welcome-file-list>
◎ 然后撤销权限过滤的注释：

<filter-mapping>
<filter-name>Au Filter</filter-name>
<url-pattern>*.do</url-pattern>
</filter-mapping>
<filter-mapping>
<filter-name>Au Filter</filter-name>
<url-pattern>*.jsp</url-pattern>
</filter-mapping>
◎ 以及：

<filter>
<filter-name>Au Filter</filter-name>
<filter-class>venus.authority.filter.FilterServlet</filter-class>
</filter>
◎ 重新启动服务后首页即为登录页面。
登录后如需在导航菜单挂接自定义页面、应用系统或组建库，可参考帮助文件的“5.4.2.4菜单管理”章节。菜单挂接相应的URL请参考项目ｗｅｂ服务内的modules.xml文件。
